## 1. 网站设置

**后台位置：** 内容管理 → 网站设置

### 📝 文字内容

| 字段 | 说明 | 页面呈现位置 | 建议 |
|------|------|-------------|------|
| **网站名称** | 显示在浏览器标签、页脚等位置 | 浏览器标签页标题、Footer 品牌名 | 建议 10 字以内 |
| **品牌名称** | 可选，若与网站名称不同则单独填写 | Header Logo 旁、Footer | 建议 10 字以内 |
| **品牌标语** | 简短宣传语 | 可用于页头或 banner | 建议 20 字以内 |
| **网站简介** | 网站整体描述 | Footer、结构化数据 | 建议 50-100 字 |
| **页脚简介** | 显示在页脚的品牌简介 | Footer 品牌描述区 | 建议 30-50 字 |
| **ICP 备案号** | 如：京ICP备12345678号 | Footer 底部 | 按实际备案号填写 |
| **版权信息** | 如：© 2024 康贝儿 版权所有 | Footer 底部 | 自动年份 + 品牌名 |
| **主联系电话** | 如：400-123-4567 | Header、Footer、浮动按钮 | 标准电话格式 |
| **主微信号** | 客服微信号 | Footer 联系方式 | - |
| **主联系邮箱** | 联系邮箱 | Footer（如有） | 标准邮箱格式 |
| **营业时间** | 如：周一至周五 9:00-18:00 | Footer 联系方式 | - |

### 🖼️ 图片内容

| 字段 | 后台操作位置 | 页面呈现位置 | 建议尺寸 |
|------|-------------|-------------|---------|
| **网站 Logo** | 内容管理 → 网站设置 → 网站 Logo | Header 左上角品牌名旁 | **200×56px**，PNG 透明底 |
| **Favicon** | 内容管理 → 网站设置 → Favicon | 浏览器标签页小图标 | **32×32px 或 64×64px**，PNG/ICO |
| **微信咨询二维码** | 内容管理 → 网站设置 → 微信咨询二维码 | Header 导航栏"微信咨询"弹窗 | **400×400px**，PNG/JPG |

### ⚙️ CTA 区块配置（所有页面底部）

| 字段 | 说明 | 页面呈现位置 | 建议 |
|------|------|-------------|------|
| **CTA 标题** | 咨询转化区块的主标题 | 所有页面底部 CTA 区块标题 | 建议 15 字以内 |
| **CTA 描述** | 咨询转化区块的描述文字 | 所有页面底部 CTA 区块描述 | 建议 30-50 字 |
| **CTA 主按钮文字** | 如：免费咨询方案 | 所有页面底部 CTA 主按钮 | 建议 8 字以内 |
| **CTA 次按钮文字** | 如：立即电话沟通 | 所有页面底部 CTA 次按钮 | 建议 8 字以内 |
| **CTA 电话号码** | 底部咨询区块显示的电话号码 | 所有页面底部"或直接拨打"后 | 标准电话格式 |

### 🔗 导航配置

| 字段 | 说明 | 页面呈现位置 | 建议 |
|------|------|-------------|------|
| **顶部导航** | 页头导航菜单 | Header 导航栏 | 建议 5-7 个 |
| **底部导航** | 页脚导航菜单 | Footer 导航区 | 建议 4-6 个 |

### 🔍 SEO 设置

| 字段 | 说明 | 页面呈现位置 | 建议 |
|------|------|-------------|------|
| **默认 SEO 设置** | 未单独设置 SEO 的页面将使用此默认值 | 搜索引擎结果 | - |

## 4. Banner 管理

**后台位置：** 内容管理 → Banner 管理

> 💡 每个页面都可以设置独立的 Banner，支持 PC 端和移动端不同图片。

### 📍 各页面 Banner 操作位置

| 页面 | 后台操作位置 | 说明 |
|------|-------------|------|
| **首页 Banner** | 内容管理 → Banner 管理 → 首页 Banner | 首页顶部大图 |
| **服务页 Banner** | 内容管理 → Banner 管理 → 服务页 Banner | 服务列表页顶部 |
| **优势页 Banner** | 内容管理 → Banner 管理 → 优势页 Banner | 优势页顶部 |
| **科普页 Banner** | 内容管理 → Banner 管理 → 科普页 Banner | 科普中心顶部 |
| **联系页 Banner** | 内容管理 → Banner 管理 → 联系页 Banner | 联系页顶部 |

### 🖼️ Banner 字段说明

| 字段 | 后台操作位置 | 页面呈现位置 | 建议 |
|------|-------------|-------------|------|
| **标题** | Banner 编辑页 → 标题 | Banner 区域主标题 | 建议 10-20 字 |
| **副标题** | Banner 编辑页 → 副标题 | Banner 区域副标题/小标签 | 建议 10-20 字 |
| **描述** | Banner 编辑页 → 描述 | Banner 区域描述文字 | 建议 30-60 字 |
| **PC 端图片** | Banner 编辑页 → 桌面端图片 | 桌面端显示的 Banner 背景图 | **1920×600px**，JPG/PNG |
| **移动端图片** | Banner 编辑页 → 移动端图片 | 移动端显示的 Banner 背景图 | **750×600px**，JPG/PNG |
| **按钮文字** | Banner 编辑页 → 按钮文字 | Banner 主按钮 | 建议 4-8 字 |
| **按钮链接** | Banner 编辑页 → 按钮链接 | Banner 主按钮跳转 | 有效 URL |
| **次按钮文字** | Banner 编辑页 → 次按钮文字 | Banner 次按钮 | 建议 4-8 字 |
| **次按钮链接** | Banner 编辑页 → 次按钮链接 | Banner 次按钮跳转 | 有效 URL |
| **位置** | Banner 编辑页 → 位置 | 决定 Banner 显示在哪个页面 | 选择对应页面 |
| **是否启用** | Banner 编辑页 → 是否启用 | 开关控制是否显示 | 开关 |
| **SEO 图片** | Banner 编辑页 → SEO 图片 | 搜索引擎分享图片 | **1200×630px**，JPG/PNG |

## ⚠️ 注意事项

1. **图片优化：** 上传前建议压缩图片（推荐 [tinypng.com](https://tinypng.com)），确保加载速度
2. **文字长度：** 遵循建议字数，避免影响页面布局
3. **即时生效：** 修改保存后，网站会在 1 分钟内更新（有缓存）
4. **SEO 设置：** 每个内容都可以单独设置 SEO，不设置则使用默认值
5. **排序规则：** 数字越小越靠前，建议使用 10、20、30... 便于插入
6. **图片格式：** 照片用 JPG，需要透明底的用 PNG，图标用 PNG 透明底
