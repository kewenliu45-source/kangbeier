# 分享图片说明

## 需要上传的文件

请在此目录上传一个名为 `share.jpg` 的分享图片：

- **文件名**: `share.jpg`
- **尺寸**: 800×800 像素（推荐）或 500×500 像素
- **格式**: JPG 或 PNG
- **用途**: 微信分享、社交媒体分享时显示的封面图

## 图片内容建议

建议包含：
- 品牌名称：好孕生命中心
- 简短标语：专业助孕咨询服务
- 品牌 Logo（如有）

## 上传方式

1. 准备好图片文件
2. 命名为 `share.jpg`
3. 放置在此目录 (`public/images/`)
4. 提交代码并推送
